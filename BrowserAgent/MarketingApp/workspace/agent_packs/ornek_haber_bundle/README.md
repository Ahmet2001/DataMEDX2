# Ornek Haber Bundle

Bu pack, Agent Studio import akisini gostermek icin hazirlanmis basit bir ornektir.

Icerik:

- `tools/haber_basliklarini_say.py`: haber basliklarindaki tekrar eden kelimeleri sayan custom tool
- `agents/haber_izleyici.yaml`: bu tool'u kullanan config agent
- `prompts/haber_izleyici.md`: agent system prompt'u

Akis:

1. Agent Studio'da pack yolunu sec
2. Preview ile manifesti kontrol et
3. Install ile tool'u custom_tools altina, agent'i agents.yaml icine bagla
